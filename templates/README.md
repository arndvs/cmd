# templates/

Reusable document templates for common operations. Copy, fill, drop into the right directory.

## Files

- `meeting-note.md` — Quick meeting capture: attendees, decisions, action items
- `decision.md` — Structured decision record: context, options, rationale

## Rules

- **Templates are starting points, not contracts.** Edit sections that don't fit.
- **Don't template everything.** If you use it less than 3 times, it doesn't need a template.
